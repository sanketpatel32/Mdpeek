# Releases

Pre-built mdpeek binaries. Generated by `npm run tauri:build` via
`scripts/copy-release.js`.

| File | Description |
| --- | --- |
| `mdpeek-<version>-setup.exe` | NSIS GUI installer (installs to `C:\Program Files\mdpeek\`) |
| `mdpeek-<version>-portable.exe` | Standalone binary — run directly, no install |

## Current

- **0.0.2** — see [CHANGELOG.md](../CHANGELOG.md) for what's new.

To install: double-click the `-setup.exe` and follow the wizard. Requires
Windows 10/11 (WebView2 is bundled with the OS).
