Tips
  • Drag & drop files onto the window to open them as tabs.
  • Export a markdown doc to a self-contained .html via the toolbar download button.
  • The find bar searches across view, edit, and PDF modes.
  • PDF tabs have a drawing toolbar — pen, highlighter, and eraser.
  • Code files (.js, .py, .json, etc.) open read-only with syntax highlighting.
  • Image files (.png, .jpg, .gif, .webp, .svg, .bmp, .ico, .avif) open in a read-only image viewer. Click to toggle fit-to-window vs actual size.
  • Wiki-links like [[Note]] or relative links like [readme](README.md) open the linked file as a new tab when clicked.
