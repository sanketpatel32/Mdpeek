  • Drag & drop files onto the window to open them as tabs.
  • Export a markdown doc to a self-contained .html via the toolbar download button, or a .pdf via the print button.
  • The find bar searches across view, edit, and PDF modes.
  • PDF tabs have a drawing toolbar — pen, highlighter, and eraser.
  • Image files (.png, .jpg, etc.) can be annotated: click the pen icon in the toolbar to draw, then save the annotated copy as a new PNG.
  • Code files (.js, .py, .json, etc.) open with syntax highlighting. Press Ctrl+E to edit them.
  • CSV/TSV files open in a sortable, filterable table view.
  • Right-click a folder in the explorer to search across every file inside it.
  • Pin frequently-used tabs via right-click → Pin tab. Pinned tabs survive app restarts.
  • Present any markdown doc as a slideshow: separate slides with --- on its own line, then click the Present button (or use the command palette). Press S to switch between deck and reading styles.
  • Live collaboration: click the Share button to get an invite link. Anyone with mdpeek installed + that link can edit your document in real time (no accounts, end-to-end encrypted via WebRTC). Both peers see each other's cursor. Works on most home networks; corporate VPNs/firewalls may block P2P.
  • Wiki-links like [[Note]] or relative links like [readme](README.md) open the linked file as a new tab when clicked.
